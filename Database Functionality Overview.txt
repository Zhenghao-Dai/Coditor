DATABASE FUNCTIONALITY

SQL Tables:
1) 'Document'
2) 'UserAccount'
3) 'Master'

Document Format:
[ docID | docName | docHost ] where docID & docHost are ints while docName is a string. Holds each document and its host.

UserAccount Format:
[ userID | userEmail | userPW ] where userID is an int and userEmail and userPW are strings. Holds each user and their credentials.

Functions:
1) addUser(params)
2) removeUser()
3) authenticateUser()
4) addDocument()
5) deleteDocument()
6) shareDocument()
